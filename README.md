# ModPacker
MapleStory Wolrds .mod file packing/unpacking tools

메이플스토리 월드 .mod 파일 패킹/언패킹 도구

# Quick Quide
1. Clone the repo.
2. Execute "out/build/x86-release/ModPacker/ModPacker.exe".
3. Input .mod file path or directory path contains vaild sources.
4. Done.
